# Project Title

Open Source Meet Manager for creating and hosting track and field meets.

## Getting Started

1. Clone this repository
2. cd meet-manager
3. Change Database Credentials in setup_db.sh, server.js
4. chmod +x ./setup_db.sh
5. ./setup_db.sh
5. npm install
6. node server.js

### Prerequisites

1. Install MySQL
2. Install Node.js
